utils¶

class sysdynpy.utils.SubclassOnlyABC(*args, **kwargs)[source]¶

Helper class to prevent instantiation. The class SysDynPy is declared as abstract and should not be instantiated. However, because there are no abstract methods present, it could be instantiated. By deriving SysDynPy from this class we ensure that no instances can be created. This is done by overriding the __new__ method. For more info see: https://stackoverflow.com/questions/50099600/abstract-classes-without-abstract-methods-creating-objects-in-python
